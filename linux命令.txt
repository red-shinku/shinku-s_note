ctrl + alt + t 打开终端

vim [filename] : 打开并编辑文件。i进入输入模式，esc退回命令模式，：进入末行模式。
	在末行模式下，w保存，w [新文件名]另存为，wq 保存退出，wq!强制（无权限写入用）
			r [filename]将文件内容导入光标位置，!命令 执行系统命令，r!命令 执行系统命令并导入结果
	在命令模式下：gg/G移动到文件头/尾, ^/$移动到行头/尾，dd/ndd删除单/n行，p粘贴，yy/nyy复制单/n行
			u撤回，ctrl+r反撤回，r替换光标所在处字符

chmod : 控制用户对文件的权限 (+增加权限，-去除权限，=重设)(x可执行，r可读，w可写）
su : 切换用户
sudo : 以root权限执行命令
sh : 执行shell命令或脚本 sh [选项] [脚本文件] [参数]
cat : 查看文件内容
nano : 编辑文本
touch : 新建文件
mkdir : 新建文件夹
rm : 删除文件
cd : 进入文件夹， ..返回上层
echo : 显示文本或变量的值
export : 设置或显示环境变量，一般写在配置文件里面
xdg-open : 打开pdf
chsh : 更改终端

./configure : 检测平台环境，在使用的系统上准备软件的构建环境，生成Makefile
            configure是一个shell脚本文件，对于使用autotools的源码包，可用autoreconf -i来生成该脚本。
make : 依据Makefile文件进行编译
make install : 安装。添加prefix=/...指定位置
make clean : 删除临时文件
make uninstall : 卸载
