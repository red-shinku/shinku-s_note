安装编译python需要的工具和库：
sudo apt install -y build-essential libssl-dev zlib1g-dev \
    libncurses5-dev libbz2-dev liblzma-dev libreadline-dev \
    libsqlite3-dev wget curl llvm tk-dev libffi-dev \
    libgdbm-dev libnss3-dev

配置编译参数：
./configure --prefix=/usr/local/python3 --enable-optimizations
./configure → 生成 Makefile（编译说明文件），根据你系统的环境做适配
--prefix=/usr/local/python3 → 指定安装路径（最终的 python3 会放在 /usr/local/python3/bin/）
--enable-optimizations → 启用优化（会让编译时间更久，但运行更快

编译：
make -j$(nproc)
make → 根据 Makefile 编译源码，生成二进制文件
-j$(nproc) → 使用 CPU 全部核心加速编译（nproc 会返回 CPU 核心数）

安装：
sudo make install
把编译好的文件安装到 --prefix 指定的路径（需要 root 权限）

配置环境变量：
echo 'export PATH=/usr/local/python3/bin:$PATH' >> ~/.bashrc
把 Python 路径追加到环境变量配置文件 ~/.bashrc，这里加到了最前面，保证优先使用刚刚装的python
source ~/.bashrc → 立即加载修改后的配置
